# rrrhody
rrrhody.org

### Installation

- Install [node.js](http://nodejs.org)
- `$ npm install -g bower`
- `$ npm install -g gulp`

### Initial Setup

- `$ cd </folder>/`
- `$ bower install`
- `$ npm install`

### Run Dev

- `$ gulp`

### Deploying

- To deploy use [ship](https://github.com/carrot/ship#usage) Github Pages deployer.
